Controle de estoque de leveduras | Coin Brewery

❮ ❯

Ferramenta desenvolvida para auxiliar o laboratório a controlar o estoque de leveduras vivas para produção de cerveja artesanal

O software foi idealizado para facilitar o controle do histórico dos processo de repicagem. Também pela necessidade de facilidade de acesso as informações críticas sobre cada uma das cepas

Obs: O site é proprietário portanto não posso permitir acesso, mas posso disponibilizar o código para visualização
